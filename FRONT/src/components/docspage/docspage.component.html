Documentation d'Installation

Suivez ces étapes pour installer et lancer le projet **HeartSenseAI**.

Étape 1 : Cloner le repository 

git clone https://github.com/gabriel25115CG/HeartSenseAI.git

Étape 2 : Accéder au dossier FRONT 

cd HeartSenseAI
cd FRONT

Étape 3 : Installer les dépendances du frontend 

npm install

Étape 4 : Revenir au dossier principal 

cd ..

Étape 5 : Accéder au dossier API 

cd API

Étape 6 : Supprimer les modules de node existants 

sudo rm -r node_modules

Étape 7 : Installer les dépendances du backend 

npm install

Étape 8 : Lancer le backend 

node index.js
