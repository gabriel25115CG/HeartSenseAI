Installation Guide

Follow these steps to install and launch the **HeartSenseAI** project.

Step 1: Clone the repository

$ git clone https://github.com/gabriel25115CG/HeartSenseAI.git

Step 2: Navigate to the FRONT folder

$ cd HeartSenseAI && cd FRONT

Step 3: Install frontend dependencies

$ npm install

Step 4: Return to the main folder

$ cd ..

Step 5: Navigate to the API folder

$ cd API

Step 6: Remove existing node modules

$ sudo rm -r node_modules

Step 7: Install backend dependencies

$ npm install

Step 8: Launch the backend

$ node index.js
